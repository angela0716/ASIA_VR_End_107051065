■ばぐまるゴシックとは？

ばぐ★ＮＡＧＡが 1996 年頃からコツコツ作っていた X680x0 用ビットマップフォントを、
Windows・macOS でも使えるようにしたものです。
全体的に形を丸く、プログラミングでも使えるよう 0(ゼロ)と O(オー)などが区別できるように作りました。
なお、漢字部分は平木敬太郎氏の「秋葉原 16」をベースに変更・追加しています。

収録フォントは以下の 3 つです。

1.「ばぐまるゴシック」
  等幅フォントで、コーディングなどに使えます。

2.「ばぐまるＰゴシック」
  プロポーショナルフォント(可変幅フォント)で、出力ウィンドウの文字など、
  できるだけ詰めて表示したい部分に使えます。

3.「BugMaru UI Gothic」
  2.と同じプロポーショナルフォントですが、コンピュータの設定画面などで使用するために
  さらに一部の文字を調整したものです。

いずれもビットマップのみのフォントですので、画面表示用としてご使用ください。

■特徴

・画面表示用・等幅ビットマップフォント
・コーディングで使えるように、0 O (ゼロとオー)、1 I l | (イチとアイとエルと縦棒)、
  , . (カンマとドット)、: ; (コロンとセミコロン)などが区別しやすい
・収録されているサイズは 4x8・8x8、6x12・12x12、8x16・16x16(半角・全角、数値の単位はピクセル)。
  ただし、推奨サイズは 8x16・16x16 ピクセル(12 ポイント)
・文字のスタイルは標準・太字(ボールド)。斜体(イタリック)は OS により自動生成
・プロポーショナルフォント(2 種類)のおまけ付き

■収録ファイル

収録されているファイルと対応するフォントは、以下の通りです。

BugMaruGothic.ttc   … ばぐまるゴシック   ／ ばぐまるゴシック Bold
BugMaruPGothic.ttc  … ばぐまるＰゴシック ／ ばぐまるＰゴシック Bold
BugMaruUIGothic.ttc … BugMaru UI Gothic  ／ BugMaru UI Gothic Bold

以下の説明では、これらをまとめて「当フォント」と呼びます。

■フォントサイズ・スタイル

次のスタイルのフォントが収録されています。

●フォント
・ばぐまるゴシック   … 等幅フォント
・ばぐまるＰゴシック … プロポーショナルフォント(可変幅フォント)
・BugMaru UI Gothic  … プロポーショナルフォント(可変幅フォント)

●サイズ
・縦  8 ピクセル (半角 4× 8 ピクセル・全角  8× 8 ピクセル)
・縦 12 ピクセル (半角 6×12 ピクセル・全角 12×12 ピクセル)
・縦 16 ピクセル (半角 8×16 ピクセル・全角 16×16 ピクセル)

  ※「ピクセル」は「ドット」ともいいます。

●スタイル
・標準 (Regular)
・太字 (Bold)

  ※ macOS では太字フォントは使用できません。

■表示のされ方

●ビットマップ表示

Windows では、画面に表示する際に次のサイズを指定すると、ビットマップ形式での表示となります。
これ以外のサイズや、収録されていないスタイル(斜体等)を指定すると、アウトライン表示となります。

・ 6 ポイント
・ 9 ポイント
・12 ポイント
※ Windows 標準の dpi 設定(96dpi)の場合。

macOS では、ビットマップではなくアウトライン表示となります。

●アウトライン表示

当フォントに収録されているビットマップフォントは前述のサイズ・スタイルのみのため、それ以外の
サイズ・スタイルでまったく表示されなくなることを避けるためにアウトラインデータも収録しています。

アウトラインデータは縦 16 ピクセルフォントの輪郭をアウトライン化したもので、拡大するとそれぞれの
ピクセルが大きくなります(カクカクした表示となります)。

環境によってはにじんだ表示になる等、きれいに表示されない場合がありますのでご注意ください。

■文字コードと収録文字

文字コードは Unicode で作成していますが、Unicode に含まれるすべての文字が収録されているわけでは
ありません。
収録されていない文字は、動作環境やアプリケーションによって表示が異なり、特定の文字("?" や "・"等)
が表示されるか、まったく表示されないか、もしくは別のフォントで表示されます。

また、縦 16 ピクセルよりも小さいサイズのフォントは読みにくくなってしまうため漢字等では形を丸く
していません。

収録されていない文字は、希望があれば今後の更新で追加していきたいと思います。
同様に、小さいサイズのフォントで形が崩れている文字もあるので、徐々に修正していきたいと思っています。
ただし、すぐに反映できるとは限りませんので、あまり期待はしないでください。

ばぐ★ＮＡＧＡが作成した文字は半角文字と全角ひらがな・カタカナ・記号です。
漢字は平木敬太郎氏の「秋葉原 16」 X680x0 版をベースにしていますが、一部修正や追加をしている文字が
あります。

■使用方法

●インストール

すでに同じフォント名のものがフォントフォルダにインストールされている場合は一度削除しないと
インストールできないことがありますのでご注意ください。

【Windows の場合】

アーカイブファイル(圧縮ファイル)を任意のフォルダに展開し、格納されている拡張子 ttc のファイルを
Windows のフォントフォルダ(コントロールパネル内のフォントフォルダ、または Fonts フォルダ)に
ドラッグ＆ドロップすることでインストールできます。

Windows Vista の場合、展開するフォルダの場所によってはインストールできないことがあります。
その場合は一度デスクトップや C ドライブ(システムドライブ)にコピーし、そこからフォントフォルダに
ドラッグ＆ドロップしてみてください。

他の方法として、展開したフォントファイルを選択し、右クリックメニューから「インストール」を
選択してもインストールすることができます。

【macOS の場合】

アーカイブファイルを任意のフォルダに展開し、格納されている拡張子 ttc のファイルをダブルクリック
すると、Font Book アプリケーションでプレビュー表示されます。
その画面で「インストール」を選択するとインストールされます。

他の方法として、ライブラリ(Library)フォルダ内の Fonts フォルダにドラッグ＆ドロップすることでも
インストールできます。

●アンインストール

実行しているすべてのアプリケーションが当フォントを使用していないことを確認してから、各 OS の
フォントフォルダから当フォントを削除してください。

■ばぐまるＰゴシックと BugMaru UI Gothic の違い

ばぐまるＰゴシックと BugMaru UI Gothic は同じ可変幅フォントですが、以下の違いがあります。

・ばぐまるＰゴシック
  ばぐまるゴシックをベースに、半角英数記号、全角英数・ひらがな・カタカナ・一部の記号について、
  文字の余白を詰めたものです。

・BugMaru UI Gothic
  主にコンピュータの設定画面(ダイアログ)に使用できるように調整したフォントです。
  ばぐまるＰゴシックをベースに、全角英数・ひらがな・カタカナ・一部の記号の文字を編集して幅を
  狭くしています。また、半角の A～Z・a～z の幅を調整しています。これは、Windows では
  半角 A～Z・a～z の幅の平均値でダイアログのサイズを決めているためです。
  そのため、半角 M や W などの幅がばぐまるＰゴシックに比べて少し広くなっています。
